Marketing Hakkında

Angular ve Nodejs kullanılarak geliştirilmiş bir Full-Stack Web uygulamasıdır. Gelen veriler liveapi şeklinde günlük olarak değişmekte olup istenilen veriler PostgreSQL DB ve Cache içinde tutulur.

Conrainer kullanılmış olup, Redis -> Docker ile kaldırılmıştır. Giriş işlemi sırasında local storage içinde validasyon işlemlerine uyan kişi verileri şifreleme ile tutulur. Bu sayede çıkış işlemi yapılmadığı sürece siteyi kapattığınızda veya refresh ettiğinizde kaldığınız sayfadan işlem yapabilirsiniz.

Open-Source bir projedir. Swagger dökümantasyonuna tıklayıp Api testlerini yapabilir, Github sayfasından kodları inceleyebilirsiniz.
